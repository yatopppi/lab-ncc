## setup

```
  git clone https://github.com/yatopppi/lab-ncc.git
  cd lab-ncc/tugas1
  sudo docker compose up -d --build 
```

cek endpoint 
```
  http://103.93.129.221:5000/
  http://103.93.129.221:5000/health
```

matiin

```
  sudo docker compose down
```
